Power & Battery Issues

Please select one or more symptoms your mouse is experiencing:

Symptom(s):

Solutions

Important: You must attempt and check at least three (3) solutions below to contact us for support.

  • Verify Windows is up to date.

  • Update the firmware on your mouse if it is on the list.

  • Verify the mouse is fully charged (use new batteries or charge Lithium-Ion batteries for 30+ minutes).

    Mouse uses a Lithium-Ion Battery Pack

    1. Turn off your Razer Mouse and charge for at least 30 mins.
    2. When charging using a Razer Mouse Dock. Clean the pogo pins and the mouse charging pins using a Q-tip and alcohol. See How to clean Razer devices.
    3. Verify if the issue is present to both charging methods (i.e. charging via a dock or USB charging cable). If issue is present on both charging methods, verify if the issue is present with another PC.
    4. If the issue is present with another PC, it is most likely an issue with the mouse. Contact support for warranty options.

    Mouse uses AA Alkaline Batteries

    1. Replace the batteries with new ones and confirm if that resolves the issue.
    2. Verify if the issue is present with another PC using the new set of batteries.
    3. If the issue is present with another PC, it is most likely an issue with the mouse. Contact support for warranty options.

  • If charging via a mouse dock, clean the pogo pins and the mouse charging pins using a Q-tip and alcohol.

  • Test your mouse on another PC and verify if the issue still exists.
