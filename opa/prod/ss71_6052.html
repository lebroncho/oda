Make sure you've entered the code properly with the correct characters. For example, "0" can either be the number zero or the uppercase letter "o", and "I" can either be the uppercase letter "i" or the lowercase letter "L".

I lost my activation code

If you have previously used the activation code, you do not need to reactivate. You can log into 7.1 Surround Sound with the Razer ID email that was used to activate the code.

I over-scratched my code activation card

You can still get an activation code by email when you've lost or over-scratched your activation code card if you register your eligible Razer device.
