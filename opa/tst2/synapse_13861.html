Application Issues

Razer device is not detected in Razer Synapse 3

Solutions

Important: You must attempt and check all solutions below to contact us for support.

  • Check if the device is working fine. Does it work normally outside of Razer Synapse 3?

  • Check in Windows Device Manager, is the device showing up and having no errors?

  • Disconnect and reconnect the device to another USB port, directly to the computer instead of a USB hub.

  • Ensure that all Razer drivers are updated through Windows Update, including optional updates.

  • Ensure that you have updated Razer Synapse 3.

  • Ensure that you have tried and followed all the steps in the Device Detection Troubleshoot Tool.

Solutions

Important: You must attempt and check all solutions below to contact us for support.

  • Check if the device is working fine. Does it work normally outside of Razer Synapse 3?

  • Disconnect and reconnect the device to another USB port, directly to the computer instead of a USB hub.

  • Factory reset the device in Razer Synapse 3 settings.

  • Ensure that you have done a clean re-installation of Synapse 3.

Crashing issue with Razer Synapse 3

Solutions

Important: You must attempt and check all solutions below to contact us for support.

  • Ensure that you have updated Razer Synapse 3.

  • Ensure that all Razer drivers are updated through Windows Update, including optional updates.

  • Ensure that there are no system admin restrictions or firewall restrictions blocking Synapse 3.

  • Ensure that you have restarted all Razer services and Razer Synapse 3.

  • Ensure that you have done a clean re-installation of Synapse 3.

Chroma RGB Issues

Device not lighted with Razer Synapse 3 on

Solutions

Important: You must attempt and check all solutions below to contact us for support.

  • Ensure that you have updated Razer Synapse 3.

  • Ensure that Windows Dynamic Lighting has been disabled for the Razer device.

  • Ensure that the brightness is enabled and set to more than 50%.

  • Try switching between Quick Effects and Advanced Effects.

  • Ensure that you have done a clean re-installation of Synapse 3.
